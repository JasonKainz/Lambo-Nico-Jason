Sesto Elemento

Der Name "Sesto Elemento" bedeutet übersetzt sechstes Elemnet, was auf das sechste Element im Periodensystem hinweisen soll, das Element ist Kohlenstoff.

Außerdem besteht die Außenhaut des Elemento überwiegen aus Kohlenstoffaser auch Carbon genannt



Der Sesto Elemento ist ein zweisitziger Sportwagen mit einem 5,2 Litet V10 Motor der eine Leistung von 570 PS schafft. Das maximale Drehmoment ist 539 Nm.

Ein PS einspricht 1,75 Kg

Das Auto schafft eine Höchstgeschwindigkeit von 350 Km/h und beschleunigt von 0 auf 100 in 2,5 Sekunden
